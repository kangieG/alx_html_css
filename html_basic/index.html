MY FIRST TOPIC

A variable has a name, a data type, and a value.
There is a location in memory associated with each variable.
A variable can be called anything or be given any name.
It is considered good practice to use variable names that are relevant to the task at hand.

THE SECOND TOPIC

The left side is the variable a value is being stored in and the right side is where the variable is being accessed.
When a variable is assigned a value, the old value is written over with the new value so the old value is gone.

THIRD TOPIC

Input / Output both deal with an outside source (can be a user or another program) receiving or giving information.
An example would be assuming a fast food restaurant is a program.
